## Data status

The dataset is **fully synthetic**. It contains no real respondents, confidential company information, or proprietary market data. The records are generated from explicit probability distributions and fixed random seeds to demonstrate an end-to-end Customer Market Insights workflow.

The data must not be interpreted as a factual estimate of Indonesian FMCG consumers.

## Reproduction

Generate the raw dataset with:

```bash
python 02_Script/01_Generate_Synthetic_Data.py
```

Create construct scores, segment assignments, and model-ready processed data with:

```bash
python 02_Script/02_FMCG_Customer_Insights_Analysis.py
```

## Main data groups

- Consumer demographics and household characteristics
- Shopping channel and product-category behavior
- Monthly FMCG spending and purchase frequency
- Promotion response
- Eight three-item consumer-attitude scales
- Processed construct scores and assigned consumer segment

See `data_dictionary.csv` for variable-level definitions.
